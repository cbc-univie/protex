protex.system.IonicLiquidSystem

class protex.system.IonicLiquidSystem(simulation: Simulation, templates: IonicLiquidTemplates, simulation_for_parameters: Optional[Simulation] = None)[source]

Bases: object

This class defines the full system, performs the MD steps and offers an interface for protonation state updates.

Parameters

  • simulation – the OpenMM simulation object

  • templates – An instance of the IonicLiquidTemplates class
